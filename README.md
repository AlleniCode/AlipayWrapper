# AlipayWrapper

## 说明

AlipayWrapper 是 AlipaySDK 的帮助类库。它从 AlipaySDK 官方 Demo 中提取除 AlipaySDK.framework 和 AlipaySDK.bundle 之外的文件。

具体使用方法，参考 Example.

## 要求

iOS: 7.0 以上

## 集成

```ruby
pod "AlipayWrapper"
```
